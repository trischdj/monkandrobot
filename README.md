# Monk & Robot

An exploratory space where contemplation and computation merge.

→ [monkandrobot.com](https://monkandrobot.com)

## Stack

- Plain HTML + CSS (no build step, no dependencies)
- Hosted on Cloudflare Pages
- Domain managed via Cloudflare

## Local Development

Just open `index.html` in a browser. No server needed.

For live reload while editing, install [live-server](https://github.com/tapio/live-server):

```bash
npx live-server
```

## Deploy

Push to `main` → Cloudflare Pages auto-deploys.

## Adding a Project

In `index.html`, copy an `<article class="project">` block inside `.projects` and update the fields.
